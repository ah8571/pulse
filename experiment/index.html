Where does the time go? (AI chat history hydration)

Each run fetches a fake conversation from the local server and measures every stage separately. TTFB = server think time + network. Download = reading the response body. Parse = JSON.parse. Render = building the DOM for every message. Only Parse and Render are stages a Rust/Wasm client library could ever touch. Run each size a few times; first runs include warm-up noise.
